Технологии проектирования и средства разработки

Нашими специалистами доработан пользовательский интерфейс Share и бизнес-логика системы управления контентом Alfresco Community Edition 4.

Задача стояла заменить стандартный компонент построения выборки контента по категориям на новый. Компонент должен был позволять выбирать несколько категорий для построения выборки контента.

Доработаны возможности системы:
  • Фильтрация и поиск контента по нескольким категориям
  • В расширенном поиске добавлена возможность объединения категории по условию «И»
  • Добавлена возможность поиска по обратному условию (Условие «НЕ»: в данном случае будут показаны все документы для которых, наобот, не совпадает заданное условие поиска)
Технологии:
  • Alfresco
  • Java
  • Spring
  • FreeMarker
  • JS
